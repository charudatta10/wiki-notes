# [Wavelet Transform](Wavelet Transform.md)

The [Wavelet Transform](Wavelet Transform.md) (WT) is a mathematical tool that decomposes a [signal-processing](signal-processing.md) into different frequency components, and then studies each component with a resolution matched to its scale. Unlike the [Fourier Transform](Fourier Transform.md), which provides only frequency information, the [Wavelet Transform](Wavelet Transform.md) provides both frequency and time information, making it suitable for analyzing non-stationary [signal-processing](signal-processing.md).
