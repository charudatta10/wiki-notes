# [Quantization](Quantization.md)

[Quantization](Quantization.md) is the process of mapping input values from a large set (often a continuous set) to output values in a smaller set (often a discrete set). In digital [signal-processing](signal-processing.md), after a [signal-processing](signal-processing.md) is [sampling](sampling.md), its amplitude values are [quantization](quantization.md), meaning they are approximated to a finite set of discrete levels. This introduces [quantization](quantization.md) error or noise.
