# [Filtering](Filtering.md)

[Filtering](Filtering.md) is a core operation in [signal-processing](signal-processing.md) used to remove unwanted components or features from a [signal-processing](signal-processing.md), or to extract desired components. Filters can be designed to pass certain frequency ranges (e.g., low-pass, high-pass, band-pass) and attenuate others. They are essential for noise reduction, [signal-processing](signal-processing.md) separation, and feature extraction.
